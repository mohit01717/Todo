# Todo App

A simple Todo application built with React, Redux, Redux-Saga, and TypeScript. This app allows users to add, edit, and delete todos.

## Features

- Add new todos
- Edit existing todos
- Delete todos
- Fetch todos from an API
- Responsive design

## Technologies Used

- React
- Redux
- Redux-Saga
- TypeScript
- Styled-Components

## Getting Started
